# Generation Skill - 生成技能

## 功能说明

基于查询结果生成自然语言回答、分析报告和数据摘要。

## 生成类型

### 1. 数值回答
将查询结果格式化为易读的数值回答：

```
指标：{指标名称}
数值：{数值} {单位}
时间：{时间范围}
地区：{地区}
```

### 2. 对比分析
多指标对比的表格和结论：

```
## 对比分析

| 维度 | 指标A | 指标B | 差异 |
|------|-------|-------|------|
| ... | ... | ... | ... |

### 结论
{分析结论}
```

### 3. 趋势描述
时间序列的趋势描述：

```
## 趋势分析

- 整体趋势：{上升/下降/平稳}
- 最大增幅：{时间}，增长{百分比}
- 最大降幅：{时间}，下降{百分比}
- 预测：{未来趋势}
```

### 4. 分布描述
占比分布的描述：

```
## 分布情况

{维度A}占比最高，达到{百分比}；
其次是{维度B}，占比{百分比}；
{维度C}占比最低，仅为{百分比}。
```

## 生成原则

1. **准确性**：基于数据，不虚构
2. **简洁性**：重点突出，不冗余
3. **可读性**：使用自然语言，避免专业术语
4. **结构化**：使用列表、表格等格式化输出

## 工具使用

- **text_template** - 文本模板渲染
- **chart_generate** - 图表生成
- **report_compile** - 报告编译
