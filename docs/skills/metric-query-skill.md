# MetricQuerySkill

## 基本信息

- **Skill ID**: `metric_query`
- **功能**: 查询各类经济指标数据
- **适用场景**: 用户询问具体数值、趋势、排名、对比等数据相关问题

## 能力范围

你可以使用此 Skill 回答以下类型的问题：

| 问题类型 | 示例 |
|----------|------|
| 单指标查询 | "北京招聘薪资是多少？" |
| 趋势分析 | "近6个月招聘趋势如何？" |
| 地区排名 | "各省份招聘薪资排名" |
| 多地区对比 | "对比北京、上海、杭州的薪资" |

## 可用工具

### 1. queryMetricCurrentValue

**描述**: 查询单个指标在指定地区的当前值

**参数**:
| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| metricName | string | ✅ | 指标名称，如：招聘数量、平均薪资、企业数量 |
| regionName | string | ✅ | 地区名称，如：北京、上海、杭州、广东省 |

**示例**:
```
用户: "北京招聘薪资"
→ queryMetricCurrentValue(metricName="招聘薪资", regionName="北京")
```

---

### 2. queryMetricTrend

**描述**: 查询指标的历史趋势

**参数**:
| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| metricName | string | ✅ | 指标名称 |
| regionName | string | ✅ | 地区名称 |
| months | int | ❌ | 查询最近几个月，默认6个月，最大24个月 |

**示例**:
```
用户: "北京近6个月招聘趋势"
→ queryMetricTrend(metricName="招聘", regionName="北京", months=6)
```

---

### 3. queryMetricRanking

**描述**: 查询指标的地区排名

**参数**:
| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| metricName | string | ✅ | 指标名称 |
| topN | int | ❌ | 显示前N名，默认5，最大20 |

**示例**:
```
用户: "各省份招聘薪资排名"
→ queryMetricRanking(metricName="招聘薪资", topN=10)

用户: "企业数量前10的省份"
→ queryMetricRanking(metricName="企业数量", topN=10)
```

---

### 4. queryMetricComparison

**描述**: 对比多个地区的同一指标

**参数**:
| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| metricName | string | ✅ | 指标名称 |
| regionNames | string | ✅ | 地区名称列表，用逗号分隔，如："北京,上海,杭州" |

**示例**:
```
用户: "对比北京上海杭州的薪资"
→ queryMetricComparison(metricName="薪资", regionNames="北京,上海,杭州")
```

---

### 5. extractDimensions

**描述**: 从用户输入中提取维度信息（指标、地区等）

**参数**:
| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| userInput | string | ✅ | 用户输入的完整文本 |

**使用场景**: 当不确定用户想查询什么时，先用此工具分析

## 可查询的指标列表

| 指标名称 | 单位 | 说明 |
|----------|------|------|
| 招聘岗位数量 | 个 | 发布的招聘岗位总数 |
| 招聘薪资 | 元 | 岗位平均薪酬 |
| 招聘企业数量 | 个 | 发布招聘的企业数 |
| 新增企业数量 | 个 | 新注册企业数 |
| 注销企业数量 | 个 | 注销企业数 |
| 在营企业数量 | 个 | 在营状态企业总数 |
| 专利申请数量 | 个 | 专利申请数 |
| 政府采购金额 | 元 | 政府采购总金额 |
| 政府采购数量 | 个 | 采购项目数量 |
| 政府采购均价 | 元 | 平均采购价格 |

## 地区识别规则

系统支持以下地区名称（包括但不限于）：
- **直辖市**: 北京、上海、天津、重庆
- **城市**: 广州、深圳、杭州、苏州、武汉、成都、西安
- **省份**: 广东省、浙江省、江苏省等
- **别称**: 帝都（北京）、魔都（上海）、鹏城（深圳）、羊城（广州）

## 使用流程

```
用户输入
    ↓
1. 提取指标名称和地区名称
    ↓
2. 判断查询类型：
   - 含"排名/前N" → 调用 queryMetricRanking
   - 含"对比/比较/VS" → 调用 queryMetricComparison  
   - 含"趋势/走势/近N个月" → 调用 queryMetricTrend
   - 其他 → 调用 queryMetricCurrentValue
    ↓
3. 整合结果，生成自然语言回复
```

## 注意事项

1. **时间处理**: 未指定时间时默认查询最新数据
2. **地区省略**: 未指定地区时默认查询"全国"
3. **指标模糊匹配**: 支持同义词，如"薪资/工资/薪酬"都会匹配到招聘薪资
4. **多维度支持**: 部分指标支持学历、经济类型等维度（自动应用默认值）
