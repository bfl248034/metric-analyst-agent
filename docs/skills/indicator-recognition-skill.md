# IndicatorRecognitionSkill

## 基本信息

- **Skill ID**: `indicator_recognition`
- **功能**: 从用户输入中识别指标意图，提取指标编码和维度信息
- **适用场景**: 需要理解用户想查询什么指标、什么维度时使用

## 能力范围

此 Skill 可以：
1. 识别用户输入中的指标名称（支持同义词）
2. 提取时间维度（最新、近N期、具体日期）
3. 提取地区维度（支持别称）
4. 提取其他维度（学历、经济类型等）
5. 处理指标无匹配的情况，提供降级方案

## 可用工具

### recognizeIndicator

**描述**: 识别用户查询中的指标和维度

**参数**:
| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| userInput | string | ✅ | 用户输入的完整文本 |

**返回结果**:
```json
{
  "matched": true,
  "indicator": {
    "indicatorId": "I_RPA_ICN_RAE_SALARY_AMOUNT",
    "indicatorName": "招聘岗位平均薪酬",
    "unit": "元"
  },
  "dimensions": {
    "region": "北京",
    "time": "last:6",
    "education": "本科"
  },
  "confidence": 0.95
}
```

**识别流程**:
```
用户输入
    ↓
1. 召回阶段（三路并行）：
   - BM25全文检索（MySQL）
   - 向量相似度（Java内存）
   - 同义词倒排（HashMap）
    ↓
2. 融合排序，取Top 3候选
    ↓
3. 大模型精排，输出：
   - 选中的指标
   - 提取的维度
   - 置信度
```

## 支持的维度类型

| 维度 | 识别方式 | 示例输入 |
|------|----------|----------|
| **地区** | 名称/别称/编码 | "北京"、"帝都"、"广东省"、"各省份" |
| **时间** | 自然语言 | "最新"、"近6个月"、"2024年"、"去年" |
| **学历** | 名称 | "本科"、"硕士"、"博士"、"大专" |
| **经济类型** | 名称 | "私营"、"国有"、"外资" |

## 时间识别规则

| 输入 | 解析结果 | 说明 |
|------|----------|------|
| "最新" / "最近" | `time: "latest"` | 查询最新一期数据 |
| "近6个月" / "最近半年" | `time: "last:6"` | 查询最近6期数据 |
| "2024年" | `time: "2024-12-31"` | 年度数据取年末 |
| "2024年2月" | `time: "2024-02-29"` | 月度数据取月末 |
| 未指定 | `time: "latest"` | 默认最新 |

## 无匹配处理策略

当无法确定用户想查询的指标时，按以下策略降级：

### 策略1: 领域推荐
**触发条件**: 召回有结果但置信度 < 0.5

**处理方式**: 分析用户意图领域，推荐该领域下的相似指标

**回复示例**:
```
抱歉，我没有找到"GDP"相关的具体指标。

但根据您的描述，您可能想查询**市场主体**方面的数据：
📊 新增企业数量（反映市场活力）
📊 在营企业数量（反映经济体量）

💬 您可以这样问：
• "北京新增企业数量"
• "各省份在营企业排名"
```

### 策略2: 澄清询问
**触发条件**: 有多个候选指标（0.3 < 置信度 < 0.5）

**处理方式**: 列出Top 3让用户选择

**回复示例**:
```
您是想查询以下哪个指标？

1. 招聘岗位平均薪酬
   反映劳动力市场价格

2. 招聘市场主体数量  
   反映招聘活跃度

3. 在营企业数量
   反映市场主体规模

请回复数字 1、2 或 3 选择。
```

### 策略3: 全局引导
**触发条件**: 完全无匹配

**处理方式**: 展示所有可查询领域和示例

**回复示例**:
```
抱歉，我没有理解您想查询什么指标。

目前我可以帮您查询以下领域的数据：

📈 招聘就业
   • 招聘岗位数量、平均薪酬、招聘企业数

🏢 市场主体
   • 新增企业、注销企业、在营企业数量

💡 知识产权
   • 专利申请数量

🛒 政府采购
   • 采购金额、数量、均价

💬 试试这样问：
• "北京招聘薪资"
• "各省份企业新增排名"
• "近6个月专利趋势"
```

## 置信度阈值

| 置信度 | 处理策略 |
|--------|----------|
| ≥ 0.8 | 高置信度，直接返回结果 |
| 0.6 ~ 0.8 | 中等置信度，直接返回但可提示用户确认 |
| 0.3 ~ 0.6 | 低置信度，使用澄清询问策略 |
| < 0.3 | 极低置信度，使用领域推荐或全局引导策略 |
