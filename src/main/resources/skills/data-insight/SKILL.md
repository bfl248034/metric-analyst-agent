# Data Insight Skill - 数据洞察技能

## 元数据
- **名称**: data-insight
- **描述**: 对指标数据进行深度分析，发现异常、关联和趋势洞察
- **版本**: 2.0.0

## 何时使用

当用户需要以下类型的高级分析时使用此技能：

1. **异常检测** - "哪些地区招聘数量异常波动"
2. **关联分析** - "人才数量和专利数量有什么关系"
3. **预测建议** - "下季度招聘趋势如何"
4. **原因分析** - "为什么企业新增数量下降了"
5. **综合报告** - "生成一份人才市场分析报告"

## 分析维度

### 异常检测
- 识别同比/环比波动超过阈值的数据点
- 标记异常高/异常低的地区
- 提供可能的原因分析

### 关联分析
- 分析指标间的相关性
- 识别领先/滞后指标
- 发现潜在因果链

### 预测建议
- 基于历史趋势预测未来走势
- 提供决策建议
- 风险提示

## 输出规范

1. **结构化呈现**: 使用表格、列表等方式清晰展示
2. **数据支撑**: 每个结论都有数据支持
3. **可视化描述**: 用文字描述图表趋势
4. **行动建议**: 给出可执行的建议

## 示例

### 示例: 异常检测
**用户**: 发现招聘数量的异常地区
**响应**:
```
招聘岗位数量异常地区检测（2024年3月）：

| 地区   | 当前值  | 同比    | 状态       |
|--------|---------|---------|------------|
| 北京   | 51,600  | +12.5%  | 🟢 正常增长 |
| 上海   | 35,135  | -4.6%   | 🟡 轻微下降 |
| 广东   | 29,968  | -9.2%   | 🔴 异常下降 |

洞察: 广东地区招聘需求下降明显，建议关注当地产业政策变化。
```

### 示例: 趋势预测
**用户**: 北京招聘薪资下季度走势如何
**响应**:
```
北京招聘薪资趋势预测：

历史趋势（近6个月）：
2023-10 → 2024-03: 12,100 → 12,580 元
增长率: +4.0%
波动幅度: 较小

预测（2024年Q2）：
- 基准预测: 12,700-12,900 元
- 增长预期: +1.0% ~ +2.5%
- 置信度: 中等

影响因素:
+ 春季招聘旺季支撑
- 部分行业需求放缓

建议: 可维持当前薪酬水平，关注技术岗位薪资竞争力。
```
