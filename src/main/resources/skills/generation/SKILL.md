# Generation Skill - 生成技能

## 功能说明

基于查询结果生成自然语言回答、分析报告和数据摘要。

## 生成类型

### 1. 数值回答
将查询结果格式化为易读的数值回答：

```
{指标名称}：{数值} {单位}
时间：{时间范围}
地区：{地区}
同比：{同比变化}
```

### 2. 对比分析
多指标对比的表格和结论：

```
## 地区对比

1. {地区A}: {数值} {单位} {趋势图标}
2. {地区B}: {数值} {单位} {趋势图标}
3. {地区C}: {数值} {单位} {趋势图标}

### 结论
{分析结论}
```

### 3. 趋势描述
时间序列的趋势描述：

```
## 趋势分析

- 整体趋势：{上升/下降/平稳}
- 最高值：{时间}，{数值}
- 最低值：{时间}，{数值}
- 平均值：{数值}
```

### 4. 排名描述
排名结果的描述：

```
## 排名情况

{指标名称}前{N}名：
🥇 {地区1}: {数值}
🥈 {地区2}: {数值}
🥉 {地区3}: {数值}
...
```

## 生成原则

1. **准确性**: 基于数据，不虚构
2. **简洁性**: 重点突出，不冗余
3. **可读性**: 使用自然语言，避免专业术语
4. **结构化**: 使用列表、表格等格式化输出
5. **业务化**: 结合业务场景给出解读

## 格式规范

- 数值保留2位小数
- 百分比显示正负号，如 +12.5%
- 趋势图标：上升↗️、下降↘️、平稳➡️
- 排名图标：🥇🥈🥉

## 示例

**输入数据**:
```
指标: 招聘岗位数量
地区: 北京
时间: 2024-03
数值: 51,600
同比: +12.5%
```

**生成输出**:
```
北京2024年3月招聘岗位数量为 51,600 个，同比增长12.5%，呈现明显上升趋势↗️。

从招聘市场活跃度来看，北京地区用工需求持续旺盛，建议关注人才供给情况。
```
